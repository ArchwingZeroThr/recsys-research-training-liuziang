## 🎯 愿景和目标
通过本次暑期科研训练，我要独立完成一次**最小科研闭环**，从选题、阅读论文、复现代码到记录和汇报，全过程自己走一遍。

**个人目标：**

- **沉淀过程资产**：完整保存代码、日志、笔记、踩坑记录，让成长有迹可循；
	
- **建立知识体系**：将零散学到的 Python/PyTorch、推荐系统、论文阅读方法等整理成属于自己的结构化笔记，方便日后快速查阅；
	
- **训练科研习惯**：养成每日记录、定期复盘、主动提问的习惯，为未来深入研究打下基础；
	
- **产出可查结果**：在培训结束时，拥有一篇精读过的论文笔记、一套可运行的代码和一份可复现的实验结果截图。

---
## ⛳️ 知识空间简介

这是我为**2026年暑期科研入门培训**搭建的个人知识库，涵盖课程全程的学习材料、作业、日志和笔记。

**空间主要内容：**

- **培训课程记录**：7次课的课堂重点和课后作业（`/assignment`）；
    
- **每日学习日志**：每日进展、问题与思考（`/daily_logs`）；
    
- **系统化知识笔记**：按主题整理的笔记（`/knowledge_base`），包括 Python/PyTorch 基础、数学基础、推荐系统范式、论文阅读方法、科研工具等；
    
- **个人学习计划**：根据自身基础制定的差异化计划（`learning_plan.md`），明确本阶段要读的论文和要跑的代码。

**使用方式**：每次学习后及时记录，定期回头整理和复习，把这里当作自己的第二大脑。
