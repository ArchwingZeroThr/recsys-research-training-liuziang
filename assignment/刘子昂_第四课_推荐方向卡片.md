# 论文：LightGCN_ Simplifying and Powering Graph Convolution Network for Recommendation

#### 宽方向（Broad Direction）

- **图神经网络推荐（Graph Neural Network for Recommendation）**  
  具体聚焦于 **协同过滤（Collaborative Filtering, CF）** 场景，利用用户-物品交互图进行高阶连通性建模。
#### 细分方向（Specific Sub-direction）

- **轻量化图卷积推荐（Light Graph Convolution for Recommendation）**  
  核心是**移除传统GCN中的特征变换和非线性激活**，仅保留**邻域聚合（Neighborhood Aggregation）**，以实现更高效、更易训练且性能更优的推荐模型。

##### 2️. 至少 **2** 类方法（Methodologies）

- **基于图传播的嵌入学习方法**：  
  LightGCN 通过多层图卷积（LGC）在用户-物品二部图上传播嵌入，仅使用加权和聚合，不引入额外参数。

- **层组合（Layer Combination）策略**：  
  将每一层传播得到的嵌入进行加权求和（默认均匀权重）作为最终表示，而非仅使用最后一层，以缓解过平滑并融合高阶邻居信息。

##### 3️. 至少 **1** 种任务或数据形式（Task / Data Format）

- **隐式反馈（Implicit Feedback）下的 Top-N 推荐任务**  
  数据形式为用户-物品交互二部图（User-Item Bipartite Graph），其中每条边表示用户对物品的点击、购买等正向交互（值为1），未观察到的交互视为负例。  
  论文使用数据集：Gowalla、Amazon-Book、Yelp2018。

##### 4️. 至少 **2** 个评价指标（Evaluation Metrics）

- **Recall@20**：在排名前20的推荐列表中，命中用户实际交互物品的比例。  
- **NDCG@20**（Normalized Discounted Cumulative Gain @20）：考虑排名位置的折损累计增益，对排序质量更敏感。  
（论文采用“全排序”协议，即对所有未交互物品进行排序评估。）
